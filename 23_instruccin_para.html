2.3 Instrucción Para

Sintaxis

 Para variable <- valor_inicial Hasta valor_final Con Paso Paso1 Hacer

instrucciones

FinPara

Descripción

 El ciclo Para se utiliza generalmente para ejecutar un conjunto de instrucciones que se repiten un número de veces, establecido antes de ejecutar el ciclo.

  • Variable: es de tipo entero.
  • Valor_inicial: este puede ser un número entero o una variable entera. Indica el valor inicial de la variable en el bucle Para.
  • Valor_final: este puede ser un número entero o una variable entera. Indica el valor final de la variable en el bucle Para.
  • Paso1 : este puede ser un número entero o una variable entera. Indica el incremento o decremento de la variable en cada iteración del bucle.
     Nota: el paso  puede omitirse  cuando su valor es 1, tanto en sintaxis estricta como flexible
